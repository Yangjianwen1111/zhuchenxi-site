# 朱晨希 · 高中数学一对一教师个人主页

南京新东方大桥南路校区高中数学一对一教师朱晨希的个人网站，用于课程介绍和家长咨询。

## 内容

- 教师介绍 · 教学理念 · 服务承诺
- 课程介绍（高一/高二/高三/1V1 定制）
- 提分方法论
- 教学成果案例
- 家长答疑 FAQ
- 在线预约咨询

## 本地预览

直接用浏览器打开 `index.html`，或：

```bash
python -m http.server 5173
```

然后访问 http://localhost:5173

## 部署

托管在 GitHub，通过 [腾讯 EdgeOne Pages](https://edgeone.ai/pages) 部署，国内访问流畅。
